 # Excel-Dashboard-Project
It is a sales dashboard created from excel-2007 version. 
It's my first ever dashboard so I hope to learn more and gain more experience in creating perfect ones.
I have used pivot tables and through them I have created pivot charts which I further used in creating a sales dashboard.
I could not add slicers as it is not available in version of excel.
I have used a dataset from kaggle. 
The data was too large that I have used only a part of it.
